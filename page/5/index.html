注意：

1.在repair-host-info-add.properties文件中，仅需配置每次新增的节点信息即可，若存在已修复过的节点信息时，在下次运行“repair.sh initSingle”指令前，请清除。

2.jdk 安装在 /opt/module 下面，不允许随意删除，否则 java 环境失效。

3.数据库密码中不得包含 “@” 。

4.主机名设置不得包含 “_”，"-" 。
